# IT Infrastructure and Systems Administration (AISI)

This repository contains the practical assignments and laboratory work developed for the **Administration of IT Systems and Infrastructure (AISI)** course, part of the 4th-year curriculum in the **BSc in Computer Engineering (Information Technology)**.

The objective of this project is to document the implementation, management, and optimization of complex IT environments, focusing on scalability, security, and automation.

* **Service Monitoring:** Ensuring high availability and performance tracking of network services.
* **Security & Policy:** Implementation of robust access controls and system hardening.
